# Synent Task 3 — Login Page UI

A modern and properly aligned Login Page UI built using React and Vite.
This project was created as part of Task 3 for Synent.

The application focuses on:

* clean UI design
* responsive layout
* real-time form validation
* interactive user experience
* modern authentication-style demo flow

---

# Live Features

## Login Form

* Email field
* Password field
* Login button
* Remember Me functionality
* Forgot Password popup modal

---

# Advanced Features Added

## Real-Time Password Validation

Checks for:

* Minimum 6 characters
* Uppercase letter
* Lowercase letter
* Number
* Special character

---

## 👁 Show / Hide Password

Modern password visibility toggle using Lucide React icons.

---

## Smart Error Handling

Displays:

* Invalid email
* Invalid password
* Invalid credentials

After 3 failed attempts:

```txt
Try demo@gmail.com / Demo@123
```

---

## Remember Me

Uses localStorage to save the email address.

---

## Forgot Password Demo Modal

* Validates email
* Only accepts demo email
* Displays demo credentials for testing

---

## Success Login Screen

Shows:

* Login success state
* Animated progress bar
* Simulated redirect UI

---

## Fully Responsive

Optimized for:

* Desktop
* Tablet
* Mobile devices

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/isha-hanaan/synent-task3-loginpage-isha.git
```

---

## 2️⃣ Open Project Folder

```bash
cd synent-task3-loginpage-isha
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```
